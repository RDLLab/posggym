# POSGGym Examples

Below are a number of examples of how to using POSGGym. The code for these examples can be found in the [examples](https://github.com/RDLLab/posggym/tree/main/examples/) directory of the repository.


## Running an Environment and Agents

- [Run an environment with random agent (run_random_agents.py)](https://github.com/RDLLab/posggym/tree/main/examples/run_random_agents.py)
- [Run an environment with specific agents (run_agents.py)](https://github.com/RDLLab/posggym/tree/main/examples/run_agents.py)
- [Run all agents for all or specific environments (run_all_agents.py)](https://github.com/RDLLab/posggym/tree/main/examples/run_all_agents.py)
- [Record Video of environment (record_video.py)](https://github.com/RDLLab/posggym/tree/main/examples/record_video.py)
- [Manually control an agent with your keyboard in a chosen environment (run_keyboard_agent.py)](https://github.com/RDLLab/posggym/tree/main/examples/run_keyboard_agent.py)
- [Print IDs of available agents of posggym.agents (print_available_agents.py)](https://github.com/RDLLab/posggym/tree/main/examples/print_available_agents.py)
